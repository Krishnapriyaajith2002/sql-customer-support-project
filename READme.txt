# Customer Support Ticket Management System (SQL Project)

This project simulates a real customer support ticketing system and demonstrates:
- Relational database design
- SQL CRUD operations
- JOIN + GROUP BY analytical queries
- SLA breach detection
- Anomaly detection on missing feedback

## Technologies Used
SQL (MySQL / PostgreSQL / SQLite)

## Key Insights from Queries
- Identify high volume ticket categories
- Measure agent performance
- Detect SLA breaches (late resolutions)
- Detect anomaly tickets without feedback

## Files Included
- schema.sql → Table creation script
- sample_data.sql → Sample dataset
- analysis_queries.sql → SQL analysis scripts

## Resume Description
Designed a SQL-based Customer Support Ticket Management System with relational schema, CRUD operations, analytical performance evaluation, SLA breach monitoring and anomaly detection for data quality improvement.
